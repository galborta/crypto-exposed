A dynamic web development project with live reloading capabilities

About this Project

This is a modern web application created with Claude's assistance, featuring Express.js server and live reloading capabilities through nodemon.

The project demonstrates how AI can help in developing web projects, setting up development environments, and managing GitHub workflows.

Features

  • Clean, responsive HTML structure
  • Custom CSS styling
  • Express.js server setup
  • Live reloading with nodemon
  • Proper semantic HTML5 elements
  • GitHub workflow integration
  • Branch management and merging

Getting Started

To use this project:

  1. Clone the repository from GitHub
  2. Run npm install to install dependencies
  3. Start the server with npm start
  4. Visit http://localhost:3000 in your browser
  5. Edit files and see live updates!
